FAQ

よくあるご質問

気になることはお気軽にご相談ください。

Claude Code を一度も触ったことがないのですが大丈夫ですか？ ⌄
まったく問題ありません。このサービスは「Claude Code を触ったことがない方」を主な対象としています。環境構築・インストールから一緒に進めますので、「名前だけ聞いたことある」という状態からでも安心してスタートできます。
プログラミング未経験でも参加できますか？ ⌄
もちろんです。むしろ非エンジニアの方（Web制作者・SEO担当・事務職・営業・DX推進担当など）を積極的に歓迎しています。Claude Code は「コードを書けない人がAIで開発できる」ツールです。あなたの職種・目標に合わせたカリキュラムを個別に設計します。
AWS未経験でも大丈夫ですか？ ⌄
問題ありません。1〜2ヶ月目は Claude Code の基礎・アプリ開発を中心に進め、AWSは2ヶ月目以降から「使える範囲だけ」段階的に学びます。S3・Lambda などの最低限を実務で使えるレベルまでサポートします。
途中で解約できますか？ ⌄
はい、いつでも解約できます。月単位のプランのため、翌月分の更新前にお知らせいただければ、違約金なしで退会できます。長期縛りは一切ありません。
オンラインサポートはどのツールで行いますか？ ⌄
Google Meet を使用します。日程は毎月柔軟に調整します。チャット相談は Discord・ChatWork・Slack・LINE など、使いやすいツールで構いません。
